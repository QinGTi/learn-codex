先看一句危险的话

“帮我做个背单词工具。”

这句话不是错，但它太像“盲盒任务”。Codex（Codex，AI 编程助手）不知道给谁用、要几个单词、页面长什么样、怎样算完成，于是它只能自己猜。

模糊版

帮我做个背单词工具。

问题：没有目标细节、没有背景、没有限制、没有验收标准。

升级版

目标：做一个适合初一学生的英语单词翻牌卡。 背景：我想复习 10 个本周单词，每张卡先显示英文，点击后显示中文。 限制：只做一个 HTML 文件；不要登录；不要联网；页面要清爽绿色。 完成标准：能打开页面；能点击翻面；能切换下一个单词；至少包含 10 个单词。

优势：Codex（Codex，AI 编程助手）知道该做什么，也知道怎么检查。

今天获得 4 张能力卡

大学课程会说 Prompt 和 Context 会影响模型输出。我们先把它变成能马上用的表达方法。

P

Prompt

Prompt（Prompt，提示词）：你给 AI（Artificial Intelligence，人工智能）的任务说明。它决定 AI 从哪里开始做。

C

Context

Context（Context，上下文）：AI（Artificial Intelligence，人工智能）当前能看到的信息。信息越清楚，越不容易乱猜。

R

Constraint

Constraint（Constraint，限制）：AI（Artificial Intelligence，人工智能）做事时必须遵守的规则，例如不要联网、不要太复杂。

✓

Acceptance Criteria

Acceptance Criteria（Acceptance Criteria，完成标准）：怎样算任务完成。它像验收题，让你检查结果。

记住这 4 格

以后你不需要每次想半天。先把任务放进 4 个格子，Codex（Codex，AI 编程助手）就更容易帮到你。

1

目标

我要做什么？给谁用？它解决什么问题？

2

背景

现在有什么材料？我懂到什么程度？我的真实场景是什么？

3

限制

不要做什么？必须遵守什么？时间、技术、风格有什么要求？

4

完成标准

打开后能看到什么？点击后发生什么？怎样检查它真的完成了？

万能骨架

目标：我要做____。 背景：这是给____用的，现在我有____。 限制：不要____；必须____。 完成标准：当____时，就算完成。

让 Codex 对比好坏 Prompt

复制这段给 Codex。你要观察：同一个任务，说明越清楚，结果是不是越接近你想要的？

复制这段 Prompt

你现在是我的 Prompt 教练。请用初一学生能懂的话，帮我对比两个任务说明的效果。 差 Prompt： 帮我做个背单词工具。 好 Prompt： 目标：做一个适合初一学生的英语单词翻牌卡。 背景：我想复习 10 个本周单词，每张卡先显示英文，点击后显示中文。 限制：只做一个 HTML 文件；不要登录；不要联网；页面要清爽绿色；代码尽量简单。 完成标准：能打开页面；能点击翻面；能切换下一个单词；至少包含 10 个单词。 请输出： 1. 差 Prompt 可能导致 Codex 猜错的 4 个地方。 2. 好 Prompt 为什么更容易成功。 3. 如果要真正开始做，请先给我一个 4 步计划，不要立刻写代码。 4. 最后给我一个“学生可复制”的 Prompt 模板。

你要盯住 4 件事

  • 它有没有指出差 Prompt 的模糊点？
  • 它有没有解释 Context（Context，上下文）的作用？
  • 它有没有遵守“先计划，不写代码”？
  • 它给出的模板能不能直接复制使用？

Context 像背包

AI（Artificial Intelligence，人工智能）只能使用它当前看得到的东西。你给它的 Context（Context，上下文）越像一个整理好的背包，它越不容易翻错东西。

可以放进背包

  • 任务目的：这是学习、复习、计划，还是展示？
  • 使用者：初一学生、老师、自己，还是同学？
  • 已有材料：单词、错题、课文、时间表。
  • 偏好：简单、清爽绿色、能在浏览器打开。

不要放进背包

  • 真实姓名、住址、学校、手机号。
  • 账号、密码、验证码。
  • 同学或家人的隐私经历。
  • 让 AI 直接替你完成作业并原样提交。

完成前先验收

Acceptance Criteria（Acceptance Criteria，完成标准）不是写给 AI 看的客套话，而是写给你自己检查结果用的。

弱完成标准

做得好看一点。 功能正常一点。 适合我用。

问题：太主观，检查时说不清到底合不合格。

强完成标准

1. 页面能直接在浏览器打开。 2. 至少有 10 个单词。 3. 点击卡片能在英文和中文之间切换。 4. 点击“下一个”能换到下一张卡。 5. 页面没有明显文字重叠。

优势：每一条都能当场检查。

今天的关键句

一个好 Prompt（Prompt，提示词）不是更长，而是让目标、上下文、限制和完成标准都清楚。
